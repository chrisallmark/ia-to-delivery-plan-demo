# Add instalment calculation service

## Service

**order-service** — REST service managing the full order lifecycle — basket, checkout, payment and fulfilment handoff. Python 3.12 / FastAPI / asyncpg / PostgreSQL 15.

## Phase

Phase 2 — Gateway & Calculator

## Description

Implement `InstalmentCalculator` in `payments/calculator.py`. The calculator derives a full repayment schedule (list of payment amounts and due dates) from an order total, term length in months, and APR. The output must match the figures the Stripe gateway returns to the nearest penny — rounding discrepancies will cause silent reconciliation failures downstream (SI-04, SI-05). This is pure domain logic with no database access or external calls. Requires SI-01 to define the `InstalmentSchedule` type the calculator returns; does not depend on SI-02.
